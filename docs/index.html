🚀 Unified Test Framework

Automatically generate unit tests for multiple programming languages from Git repositories

curl -sSfL https://install.uft.dev | sh

🌍 Cross-Platform

Works seamlessly across all major platforms

🍎 macOS 🐧 Linux 🪟 Windows

🔧 Multi-Language

Supports 9+ programming languages

☕ Java 🟨 JavaScript 🐍 Python 🦀 Rust 🐹 Go

⚡ Smart Testing

Intelligent framework detection and selection

JUnit Jest pytest More...

🎯 Quick Usage

# Generate tests from any Git repository
uft git-repo https://github.com/user/awesome-project.git

# View supported languages and frameworks
uft languages

# Generate tests for a single file
uft generate src/main.js
